# Usage

1. Run backend with

```bash
cd backend
./gradlew run -t
```

2. Run frontend with

```bash
cd frontend
./gradlew wasmJsBrowserRun -t
```

3. Frontend starts on [127.0.0.1:8080](http://127.0.0.1:8080) by default

Backend data (the file that is being edited) will be saved to the `data` directory (in the working directory)
